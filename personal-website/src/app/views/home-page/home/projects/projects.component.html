DTOUR

As part of a course project, I designed the front end of a web application (DTOUR) for destination-oriented taxi and rideshare drivers in New York.
Given a source and destination, DTOUR would suggest paths with a higher chance of picking up passengers going towards the destination.
Suggested routes were, on average 25 times more likely to pick up someone with a similar route than the route suggested by Google Maps.
I implemented the front-end in Javascript using Google Maps Directions and Services API.

Scene recognition with bag of words

As part of a course project, I implemented an application that clusters quantized local features from training images to build an index of features.
Then, it uses SIFT features and linear classifiers learned by support vector machines to classify scenes into categories; kitchen, forest, office etc.
I was able to reached an accuracy of 80%.
